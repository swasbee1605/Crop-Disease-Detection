# 🌱 SmartKheti – AI-Based Crop Disease Detection

SmartKheti is a web application that helps farmers detect tomato leaf diseases using Machine Learning. Users can upload an image of a tomato leaf, and the system predicts the disease while providing treatment suggestions to support timely crop management.

## 🚀 Live Demo

Visit the application here:

👉 **https://crop-disease-detection-p1oh.onrender.com/**

**Note:** Since the application is hosted on Render's free tier, the first request after inactivity may take 30–60 seconds while the server wakes up.
---

## 📌 Features

- 🌿 AI-powered tomato leaf disease detection
- 📷 Image upload and instant prediction
- 💊 Disease-specific treatment recommendations
- 👤 User registration and authentication
- 📱 Responsive and user-friendly interface
