# Crop-Disease-Detection
This project focuses on early detection of crop diseases leveraging deep learning techniques to support smarter and more sustainable agriculture.  From initial brainstorming sessions to giving the idea structure, design, and implementation, we built an end-to-end pipeline capable of identifying diseased crop samples using image data.

Model Architecture: Combination of Convolutional Neural Networks (CNNs) and Autoencoders for robust feature extraction and classification.
Goal: Rapid and accurate detection of disease symptoms in crop images.
Accuracy Achieved: ~80%

Workflow Covered:
Problem ideation and market need analysis
Dataset exploration & preprocessing
Architecture planning
Model training, validation & tuning
Performance evaluation
Result visualization and report generation

Tech Stack:
Python, Keras, TensorFlow, OpenCV, NumPy, Matplotlib
